# 파일 정리 도우미 (File Cleaner)

> 다운로드, 바탕화면, 카카오톡 받은 파일 등 지저분한 폴더를 원클릭으로 정리해주는 웹 서비스

## 방향 전환 이력

- ~~Python 포터블 앱 (.exe)~~ → **웹앱 (File System Access API)** 으로 전환
- 이유: 설치 불필요, 사용자 진입 장벽 최소화, 수익화·배포·업데이트 모두 유리
- 기존 Python 코드 (`app.py`, `organizer.py`, `config.py`)는 프로토타입으로 보존

## 핵심 기술: File System Access API

- 브라우저에서 직접 사용자 PC 파일에 접근 (사용자 허용 필요)
- 파일이 서버로 전송되지 않음 → 개인정보 보호, 속도 빠름
- `move()` API로 파일 이동 시 디스크 I/O 없이 경로만 변경 → 대용량 파일도 즉시 처리
- 미지원 브라우저에서는 자동으로 복사+삭제 방식으로 폴백
- 크롬, 엣지 지원 (한국 브라우저 점유율 90%+)

## 현재 구현된 기능

### 파일 분류
- 12개 카테고리 자동 분류 (사진, 문서, 영상, 음악, 압축파일, 설치파일 등)
- 메신저 파일 자동 감지: KakaoTalk, WhatsApp, LINE, Telegram, WeChat, Discord, Zalo
- 스크린샷 자동 감지 (한국어, 영어, 일본어, 중국어)
- 파일명에서 날짜 추출 → 날짜별 폴더 정리 옵션

### 중복 파일 탐지 + 삭제
- 파일 크기로 1차 필터링 → SHA-256 해시로 2차 확인
- 중복 그룹별 결과 표시
- 각 그룹에서 보존할 파일 선택 → 나머지 일괄 삭제
- 삭제 전 확인 다이얼로그 + 확보 용량 표시

### 웹 UI
- 폴더 선택 → 스캔 → 미리보기 → 실행 흐름
- 카테고리별 통계 (파일 수, 용량)
- 실시간 진행률 표시
- 반응형 디자인
- **폴더 구조 미리보기**: 정리 후 폴더 구조를 트리 형태로 시각적 확인 (목록/트리 전환)
- **되돌리기(Undo)**: 정리 실행 후 원래 상태로 복원 가능 (localStorage에 기록 저장 — 창 닫아도 복원 가능)

### 성능 최적화
- **move() API 우선**: 파일 내용을 읽고 쓰는 대신 경로만 변경 — 파일 크기와 무관하게 즉시 처리
- **5개 병렬 처리**: 파일을 하나씩이 아닌 5개씩 동시에 이동/복원
- 미지원 브라우저에서는 자동으로 복사+삭제 방식으로 폴백

### 안전성
- 파일은 복사 완료 후 원본 삭제 — 도중에 중단되어도 파일 손실 없음
- 되돌리기 기록은 localStorage에 저장 — 브라우저를 닫아도 나중에 복원 가능
